# Octoblu Pebble App

Connect your Pebble watch to Skynet.im

## Prerequisites

- Git
- Python
- [Pebble SDK](http://developer.getpebble.com/2/getting-started/) installed on your computer

## Installation & Testing:

1. `git clone git@github.com:octoblu/octoblu-pebble.git`
2. `cd octoblu-pebble`
3. `pebble build`
